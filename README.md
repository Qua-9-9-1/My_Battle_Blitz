# arcade-microGames
micro games like warioware inc


controls on keyboard:
- P1 : A = C
- P1 : B = V
- P1 : C = B
- P1 : UP = Z
- P1 : DOWN = S
- P1 : LEFT = Q
- P1 : RIGHT = D
- P1 : START = TAB

- P2 : A = NUMPAD_1
- P2 : B = NUMPAD_2
- P2 : C = NUMPAD_3
- P2 : UP = UP_ARROW
- P2 : DOWN = DOWN_ARROW
- P2 : LEFT = LEFT_ARROW
- P2 : RIGHT = RIGHT_ARROW
- P2 : START = ENTER
